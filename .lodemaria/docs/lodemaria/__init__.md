# Lodemar.IA

This is a Python package designed to interact with an Ollama model for chat sessions that can search the web. It provides a command-line interface to run the application.

## Public/Exported Members

### Module Constants

- `__version__` (str): Version of the Lodemar.IA package. Currently set to "1.0.0".

## Entry Point

The entry point for running this package is:

```bash
python -m lodemaria
```

This command will execute the main script, which is typically named `lodemaria.py`.
